# AIReport

`Boost.JSON` 학습 과정을 단계별로 정리하는 폴더다.

## 구성

- `Next-Steps.md`
- `Phase-01-Parse-And-Serialize.md`
- `Phase-02-Read-Json-Types.md`
- `Phase-03-Build-Json-Programmatically.md`

## 사용 방식

문서를 아래 순서로 읽고 갱신한다.

1. 이번 주제의 전체 목표를 확인한다.
2. `Next-Steps.md`에서 다음 구현 phase를 고른다.
3. 코드를 만들고 직접 실행하거나 수정해 본다.
4. phase 문서를 추가하고 배운 점과 다음 질문을 기록한다.

## 현재 범위

현재는 아래 주제를 계획 중이다.

- parse / serialize
- JSON type 접근
- JSON DOM 직접 조립
- 사용자 정의 타입 변환
- validation / error handling
- 성능 기초
